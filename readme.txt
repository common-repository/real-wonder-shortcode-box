== Description == 
[REAL Shortcodes](http://jb-webs.com/downloads/category/plugins/short-coder/boxes/) is WordPress plugin what gives you page the extra touch to make it memorable. This is a really premium plugin that you can get absolutely for free! 

But it has more under the hood, because it comes with a set of finest CSS3 effects like:

*	Breath catching transition effects to create animate your content.
* 	Never before seen CSS 3D shadows [css -moz-box-shadow](http://jb-webs.com/downloads/category/plugins/short-coder/boxes/)  
*	Huge collection of jQuery Roll over Effects
*	GUI for CSS editing
*	WYSIWYG editor
*	JQuery Color Picker 

REAL Shortcodes is not just another plugin with a collection of boxes, icons and buttons, it is rather a plugin to create such items with a GUI (no coding required) in an easy to understand WYSIWYG environment.

== Installation ==
The plugin is simple to install:
* 	Download ` REAL-Shortcodes.zip`
* 	Unzip
* 	Upload ` REAL-Shortcodes` directory to your `/wp-content/plugins` directory
* 	Go to the plugin management page and enable the plugin

You can find some demos and inspirations under  http://jb-webs.com/downloads/category/plugins/short-coder/boxes/
